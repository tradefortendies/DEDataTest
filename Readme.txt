How to run

1. Install Matlab runtime:
https://se.mathworks.com/help/compiler/install-the-matlab-runtime.html

First way:
1. run MyAppInstaller_web.exe
3. Go to folder where you installed it and you can find test.exe from application folder
4. Throw dataScraper.py to the folder where your .exe is (if you want fresh data)
6. Run DeDataTest.exe

Second way:
1. run DeDataTest.exe which is found in for_redistribution_files_only
2. Throw dataScraper.py to the folder if you want fresh data
